KeePassX-min is a fork of KeePassXC with a stripped down to a minimum subset of features for offline password manager use on Linux.

Goal of this project is to provide safe and secure subset of features for offline password manager and keep it up to date with modern Linux dependencies, like Qt6.

### Basic features
* Create, open, and save databases in the KDBX format (KeePass-compatible with KDBX4 and KDBX3)
* Store sensitive information in entries that are organized by groups
* Search for entries
* Password generator
* Import databases from CSV, 1Password, Bitwarden, Proton Pass, and KeePass1 formats

### Advanced features
* Database reports (password health and statistics)
* Database export to CSV, XML, and HTML formats
* TOTP storage and generation
* Field references between entries
* File attachments and custom attributes
* Entry history and data restoration
* Additional encryption choices: Twofish and ChaCha20
